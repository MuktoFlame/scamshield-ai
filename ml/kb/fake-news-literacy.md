# Spotting misinformation

## Style is a warning, not a verdict
ALL-CAPS headlines, "SHOCKING", "they don't want you to know", and demands to
share before it is "deleted" are the style of engagement bait. Sober-looking
text can still be false, and emotional text can be true — style is a reason to
check, not a conclusion.

## Check the claim, not the vibe
Take the single most concrete claim in the story — a name, a number, a date —
and search for it from an independent source. If only one website in the world
carries a "huge" story, that is the answer. Real events leave traces across
unrelated outlets within hours.

## Manipulated context
The most common misinformation is not fabricated text but a real photo or clip
with the wrong caption — an old event presented as today's, a different
country presented as yours. A reverse image search usually settles it in
seconds.

## Before you share
Misinformation spreads because sharing takes one second and checking takes
one minute. If a post makes you angry or triumphant, that emotional spike is
exactly when to pause — content engineered to go viral targets feelings, not
facts.
